Welcome to the Zephyr Documentation

One Product. Three Editions. Everything You Need in One Place.

We’ve recently consolidated Zephyr Squad and Zephyr Scale into a single product family: Zephyr.

Product Overview

Zephyr is SmartBear’s native test management solution for Jira. It helps teams plan, create, execute, and track their software testing activities directly within their Jira spaces. The platform centralizes test assets and provides a unified workspace where testers, developers, and product teams can collaborate effectively.

Zephyr supports both manual and automated testing and streamlines QA workflows with built-in intelligent insights. It also includes AI capabilities that accelerate test creation, improve test coverage, and enhance overall efficiency.

With clear visibility into test quality and release readiness, and with its close integration into work items and development processes, Zephyr allows teams to manage the complete testing lifecycle without switching tools.

Key Concepts

A brief introduction to essential components within Zephyr:

  • Test Cases: Define the steps, data, and expected results for validating functionality
  • Test Cycles: Organize and group tests for execution within a release or sprint
  • Test Plans: Combine multiple cycles to manage testing across larger releases
  • Automation: Integrate CI/CD pipelines, automation frameworks, and test results
  • Integrations: Connect with tools like Jenkins, Bamboo, Cypress, JUnit, and more
