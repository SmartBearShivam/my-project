## Zephyr Editions Comparison

This page provides a side-by-side comparison of features available across different Zephyr editions.

Whether you are evaluating which edition best fits your team’s needs or want to understand what is included in your current plan, this table helps you quickly compare the differences.

### Notes

- **Zephyr Essential** provides core testing capabilities along with advanced automation features like no-code testing and parallel execution.  
- **Zephyr** expands support for cross-browser testing and maintains strong integration capabilities.  
- **Zephyr Advanced** focuses on enterprise-level needs such as scalability and broader traceability.
